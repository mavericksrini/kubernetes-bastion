# kubernetes-bastion
Docker container for managing Kubernetes-The-Hard-Way tasks for Google Compute Engine, etc.

This project creates a Centos-7 image all setup for managing Kubernetes. It was developed as
I have to use a Windows 10 laptop and needed a good approach to going through Kubernetes
the hard way step-by-step when preparing for the Kubernetes Certified Administrator (CKA)
exam. 

Basically I fire this up on Windows 10, exec into the container and manage my Kubernetes cluster
that is running on GCE.
